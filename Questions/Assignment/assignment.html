User form with following fields and requirements

Sections

  • Form

    Fields
    • First Name
    • Last Name
    • Password
    • Confirm Password
    • Email
    • Is not robot checkbox
    • Submit Button

  • User Table (List view)

Requirements

  • All fields are mandatory
  • First Name should be of at least 10 chars.
  • Name must have alphabets only.
  • Email must be valid
  • Submit button would anble only when Is not robot checkbox is checked.
  • If user enters valid data, then on click add that record to User Table (List View)
